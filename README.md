# Description
Homework 10: Template Engine is a software engineering team generator command line application. The application prompts the user for information about the team manager and then information about the team members. The user can input any number of team members, and they may be a mix of engineers and interns. When the user has completed building the team, the application creates an HTML file that displays a nicely formatted team roster based on the information provided by the user. 
    
## Installation
Clone repository to your local drive. Run npm install to install dependencies.
